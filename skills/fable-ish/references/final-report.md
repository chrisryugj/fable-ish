# fable-ish Final Report

Keep the final answer concise. Include only evidence-backed claims.

## Standard Shape

- What changed.
- Files or surfaces that matter.
- Verification commands actually run.
- Result of each verification.
- Known gaps, blockers, or unverified risk.

## Plugin Conversion Report

For fable-ish plugin work, include:

- Name retention confirmation.
- Change summary.
- Created files.
- Removed files.
- Hook roles.
- Excluded over-engineering.
- Verification commands and results.
- Remaining limitations.
- Next improvement candidates.

Do not say a hook, test, build, or validator passed unless it was run and observed.
